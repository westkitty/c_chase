# Cloud Chaser Test Ready Package

Open `index.html` first.

Files:
- `cloud_chaser_playable.html` — original playable HTML5 build from the uploaded ZIP.
- `cloud_chaser_seattle_remastered.html` — original named source build from the uploaded ZIP.
- `cloud_chaser_guide_overlay.html` — standalone generated guide overlay prototype.
- `cloud_chaser_asset_preview.html` — original non-player sprite preview.
- `cloud_chaser_snes_non_player_sprites.png` — required by the asset preview.

Known limits:
- The guide overlay is a companion/prototype, not injected into the live game runtime.
- Gameplay was not browser-playtested here; static JS syntax and source checksums were verified.
