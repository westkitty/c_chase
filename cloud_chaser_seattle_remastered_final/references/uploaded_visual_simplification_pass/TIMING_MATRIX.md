# Cloud Chaser Timing Matrix

These targets define the first release-candidate balance expectations. They are intentionally conservative until full human completion telemetry exists.

| Level | Casual clear | Practiced clear | S-rank target | Assist clear |
|---|---:|---:|---:|---:|
| PIKE PLACE 1-1 | 92s | 68s | 51s | 110s |
| SEATTLE CENTER 2-1 | 108s | 82s | 62s | 128s |
| WATERFRONT 3-1 | 103s | 78s | 59s | 124s |
| GAS WORKS BONUS | 98s | 72s | 55s | 118s |
| FREMONT BRIDGE 5-1 | 112s | 86s | 66s | 134s |
| MONORAIL FINALE | 116s | 90s | 70s | 140s |

## Reasoning

- Casual clears should leave room for mistake recovery and first-read level comprehension.
- Practiced clears should reward route knowledge, checkpoint efficiency, and clean cloud routing.
- S-rank targets should require high-route confidence, route medal progress, and low damage.
- Assist clears should remain achievable without converting Breezy Mode into an autopilot. The player still has to play. Tragic, but necessary.
