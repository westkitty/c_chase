# Controls and Accessibility Notes

## Keyboard defaults

| Action | Default binding |
|---|---|
| Move left/right | Arrow keys or A/D |
| Menu up/down | Arrow Up/Down or W/S |
| Jump / confirm | Space, Arrow Up, or W |
| Cloud blow | X or J |
| Reserve item | C |
| Breezy assist | B |
| Pause | P |
| Restart run | R |
| Level select | L |
| Options | O |
| Ghost replay | G |
| Music/sound toggle | M |
| Debug overlay | F3 |
| Practice toggle | T |

## Rebinding

The options menu now exposes rebinding for every defined action, not only jump/blow/reserve. Open with `O`, choose an action, press Enter/Space, then press the replacement key.

## Gamepad

Gameplay and menus support standard browser gamepads:

- D-pad/left stick: move and menu navigation
- Face button A/B: confirm/back style behavior
- Shoulder/trigger combinations: restart/options/level-select support

## Touch

Coarse pointer/mobile mode exposes expanded touch buttons for movement, jump/confirm, cloud blow, reserve item, level select, and options. This is still not a full mobile redesign; it is parity-enough for browser testing.

## Accessibility toggles

Options include sound, music layers, scanlines, screen shake, reduced motion, ghost replay, tutorial coach, and touch controls.
